## H-love-time

一款自适应的恋爱计时网页

![恋爱计时](https://i0.wp.com/s2.loli.net/2023/11/03/L96ZW3fgyGE5UFq.png)

### 时间配置

`js`文件夹内`love.js`设置第`17`行的时间

### 功能

-  简洁优美
-  自动明暗
-  自适应化
